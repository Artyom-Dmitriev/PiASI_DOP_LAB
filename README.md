# PiASI — Лабораторная работа (доп.)

Программа на языке **C** для работы с базой данных студентов:
динамический массив структур, ввод/вывод с клавиатуры, сортировка,
поиск, текстовые (CSV) и бинарные файлы.

Реализованы все восемь функциональных пунктов из задания:

| # | Пункт задания | Функции в коде |
|---|---|---|
| 1 | Ввод студентов с клавиатуры | `input_students` |
| 2 | Вывод таблицы | `print_table` |
| 3 | Сортировка по фамилии / группе / среднему баллу | `sort_students` (через `qsort`) |
| 4 | Поиск по номеру группы | `find_by_group` |
| 5 | Сохранение / загрузка CSV | `save_csv`, `load_csv` |
| 6 | Сохранение / загрузка бинарного файла | `save_bin`, `load_bin` |
| 7 | Добавление / удаление записи через `realloc` | `add_student`, `remove_student` |
| 8 | Корректное освобождение памяти и закрытие файлов | `free(arr)` в конце `main`, `fclose` во всех функциях I/O |

## Структура проекта

```
PiASI_DOP_LAB/
├── src/main.c        — исходный код программы
├── data/             — файлы данных, создаваемые программой (.csv, .bin)
├── build/            — собранный .exe (в Git не хранится)
└── .vscode/
    ├── tasks.json    — задача сборки (Ctrl+Shift+B)
    └── launch.json   — конфигурация отладки (F5, через gdb)
```

## Требования

- **GCC** (MinGW-w64 под Windows, либо системный gcc под Linux/macOS).
- Под Windows программа автоматически переводит консоль в UTF-8
  через `SetConsoleOutputCP(65001)`, чтобы корректно показывать
  кириллицу.

## Как собрать

В терминале из корня проекта:

```powershell
gcc -Wall -Wextra -g src/main.c -o build/lab.exe
```

Или в VS Code: **Ctrl + Shift + B**.

## Как запустить

```powershell
.\build\lab.exe
```

Или в VS Code: **F5** (запуск под отладчиком, остановка на breakpoint'ах).

## Меню

После запуска появится меню:

```
 1. Ввести список студентов с клавиатуры
 2. Показать таблицу студентов
 3. Сортировать (по фамилии / группе / среднему баллу)
 4. Найти студентов по номеру группы
 5. Сохранить в текстовый файл (CSV)
 6. Загрузить из текстового файла (CSV)
 7. Сохранить в бинарный файл
 8. Загрузить из бинарного файла
 9. Добавить запись
10. Удалить запись
 0. Выход
```

При сохранении/загрузке имя файла можно ввести или нажать Enter
для значения по умолчанию: `data/students.csv` или `data/students.bin`.

## Формат CSV

```
surname,initials,group,g1,g2,g3,g4
Иванов,И.И.,101,5,4,5,4
Петров,П.П.,102,3,4,4,5
```

Средний балл в файле не хранится — он автоматически пересчитывается
при загрузке.

## Формат бинарного файла

```
[size_t n][Student #0][Student #1] ... [Student #n-1]
```

В начале файла — количество студентов одним `size_t`,
далее — массив структур `Student` целиком, как лежит в памяти.

Бинарный файл компактнее текстового и читается одним вызовом
`fread`, что и было одним из требований задания.

## Сценарий тестирования

1. **1** → ввести 3 студентов с разными группами и оценками.
2. **2** → проверить таблицу и автоматически посчитанный средний балл.
3. **3 → 1** → отсортировать по фамилии, **2** — проверить.
4. **3 → 3** → отсортировать по среднему баллу (по убыванию).
5. **9** → добавить ещё одного студента, **2** — проверить.
6. **10** → удалить любого по номеру, **2** — проверить.
7. **4** → ввести номер группы, посмотреть найденных.
8. **5** → сохранить в CSV (Enter — путь по умолчанию).
   Открыть `data/students.csv` в любом редакторе — увидеть текст.
9. **0** → выйти, запустить снова.
10. **6** → загрузить из CSV, **2** — данные восстановились.
11. **7** → сохранить в бинарный (`data/students.bin`).
    Сравнить размеры файлов: `.bin` обычно компактнее `.csv` при
    небольшом числе студентов он может быть и больше из-за
    фиксированного размера структуры, но при росте N разрыв растёт
    в пользу бинарного.
12. **8** → загрузить из бинарного, **2** — данные на месте.

## Использованные стандартные средства

- `<stdlib.h>`: `malloc`, `calloc`, `realloc`, `free`, `qsort`.
- `<stdio.h>`: `printf`, `scanf`, `fgets`, `fopen`, `fclose`,
  `fprintf`, `fscanf`, `fwrite`, `fread`.
- `<string.h>`: `strcmp`, `strncpy`, `memmove`, `memset`, `strlen`.
- Под Windows: `<windows.h>` — только для `SetConsoleOutputCP`.
