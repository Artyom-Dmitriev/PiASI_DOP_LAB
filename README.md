# PiASI — Лабораторная работа (доп.)

Программа на языке C для работы с базой данных студентов
(динамический массив структур, ввод/вывод, сортировка, поиск,
текстовые и бинарные файлы).

## Структура проекта

```
PiASI_DOP_LAB/
├── src/main.c        — исходный код программы
├── data/             — файлы данных, создаваемые программой (.csv, .bin)
├── build/            — собранный .exe (не хранится в Git)
└── .vscode/          — настройки сборки и отладки для VS Code
```

## Как собрать

В терминале из корня проекта:

```powershell
gcc -Wall -Wextra -g src/main.c -o build/lab.exe
```

Или в VS Code: `Ctrl + Shift + B`.

## Как запустить

```powershell
.\build\lab.exe
```

Или в VS Code: `F5` (запуск под отладчиком).

## Зависимости

- GCC (MinGW-w64) — компилятор языка C.
- VS Code с расширением `C/C++` от Microsoft — для разработки и отладки.
